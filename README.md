Nextjs web project.
+ Quick styling with Tailwind.
+ FontAwesome appealing Icon Library.

Installation.

Clone the repo:

    git clone git@github.com:aflotemx/web

Navigate to the project folder:

    cd web

Install dependencies:

    npm i

Test server:

    npm run dev

Dockerize:

    docker build . -t web

Run docker server:

    docker-compose up -d
